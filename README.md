# Cloud-Hosted-College-Portal
Cloud-Hosted College Portal is a simple web application that stores and manages messages submitted by users through an online form. The project uses Firebase Firestore as the backend database and Firebase Hosting for cloud deployment. It provides an easy, secure, and scalable way to collect user messages in real time.

Features:
1.Web form for users to submit messages
2.Messages stored securely in Firestore Database
3.Real-time data retrieval and management
4.Fully deployed using Firebase Hosting
5.Lightweight and easy to integrate with other college modules

Tech Stack:
HTML, CSS, JavaScript – Frontend
Firebase Firestore – NoSQL database
Firebase Hosting – Cloud deployment
GitHub – Version control & project management

Project Structure:
/project-folder
│── index.html
│── style.css
│── script.js
│── firebase-config.js
│── README.md
│── .gitignore

How It Works:
1.User submits a message using the web form
2.The message is sent to Firestore
3.Admin can view messages from the Firebase Console
4.The entire website is hosted on Firebase Hosting

Deployment:
1.The project is deployed using:
  firebase deploy
2.You can update anytime using:
  firebase deploy --only hosting

Usage:
1.College websites
2.Feedback collection portals
3.Contact/Query submission systems
4.Lightweight cloud-based message management
